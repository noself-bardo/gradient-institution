## Purpose

Preserved Human Capability Matrix shows which human capabilities remain preserved, supported, extended, weakened, or at risk.

## Constitutional Role

It matters because The Gradient must not let systems quietly replace the human capacities they were meant to preserve.

## Boundaries

It is distinct from Capability Map because it focuses on preservation status. It is distinct from Automation Boundary because it tracks human capacity, not automation authority.

## Inputs / Outputs

### Inputs

- human capability
- preservation status

- risk condition
- review evidence

### Outputs
