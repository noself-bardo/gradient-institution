## Purpose

Practice Framework Registry records named practices, use boundaries, statuses, and review notes.

## Constitutional Role

It matters because repeatable practices need stewardship without becoming canon merely because they are listed.

## Boundaries

Practice Framework Registry is distinct from Practice Framework because it records practices but does not perform them. It is distinct from Engine Registry because it governs human practice forms rather than transformation engines.

## Inputs / Outputs

- use boundary

- steward owner

### Outputs
