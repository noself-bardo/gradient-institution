## Purpose

Trust State records the current confidence posture around material based on provenance, evidence quality, and review limits.

## Constitutional Role

It matters because The Gradient needs proportional use of material without treating trust as truth, canon, or permanent status.

## Boundaries

Trust State is distinct from Status State because it tracks confidence, not institutional workflow status. It is distinct from Validation Gate because it records posture rather than testing whether something works.

## Inputs / Outputs

### Inputs

- source trace
- evidence quality
- review limit
- contradiction note
- use context

### Outputs

- trust posture
- confidence limit
- use constraint
- review trigger
- provenance reminder

### Non-Outputs

- truth
- canon
- approval
- permanent trust
- personal reputation score
