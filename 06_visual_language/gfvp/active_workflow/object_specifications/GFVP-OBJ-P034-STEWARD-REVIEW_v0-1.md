## Purpose

Steward Review structures human review before consequential institutional movement.

## Constitutional Role

It matters because engines, prompts, status labels, and automations cannot carry final responsibility for consequential action.

## Boundaries

Steward Review is distinct from Validation Gate because it is human review, not a test gate. It is distinct from Confirmation Receipt because review evaluates; a receipt records explicit confirmation.

## Inputs / Outputs

### Inputs

- review item
- evidence trace
- risk note
- decision options
- authority boundary

### Outputs

- review finding
- decision route
- deferral note
- escalation need
- human accountability trace

### Non-Outputs

- automatic approval
- canon
- automation decision
- image approval
- ratification

## Required Labels

- Steward Review

- Risk Note
- Review Finding
- Human review required

- automatic promotion or ratification

- image gate as subject matter

- object collapse into adjacent roadmap plates
