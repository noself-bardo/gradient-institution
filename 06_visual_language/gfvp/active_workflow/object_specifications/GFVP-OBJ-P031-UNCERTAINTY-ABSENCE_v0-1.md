## Purpose

Uncertainty / Absence records what is unknown, missing, unavailable, or not yet proven.

## Constitutional Role

It matters because The Gradient must hold gaps honestly instead of filling them with narrative confidence or invented source material.

## Boundaries

It is distinct from Capability Gap because it concerns knowledge and evidence absence, not capacity. It is distinct from Conflict / Contradiction because absence is not disagreement.

## Inputs / Outputs

### Inputs

- question
- missing evidence
- known limit
- search trace
- use risk

### Outputs

- uncertainty statement
- absence marker
- search need
- use caution
- review route

### Non-Outputs

- answer
- proof
- rejection
- canon
- invented evidence
