## Purpose

Source Review examines an admitted artifact's source condition before interpretation, metabolization, validation, promotion, or canonization may occur.

It clarifies what kind of source the artifact is, how much provenance is visible, what trust limits apply, and whether the artifact is ready for later institutional work.

## Constitutional Role

Source Review protects The Gradient from treating admitted material as interpreted knowledge too early.

It preserves the distinction between entry, review, interpretation, and authority.

It also protects unknown-provenance material from false certainty by allowing it to remain visible while clearly marked.

## Boundaries

Source Review follows Admission Gate.

It is not Admission Gate.

It is not Metabolization.

It is not Validation.

It is not Promotion.

It is not Canonization.

It does not decide final truth.

It does not create institutional doctrine.

It determines whether the admitted artifact has enough source clarity for an appropriate next action.

## Inputs / Outputs

### Input

Admitted Artifact.

The artifact must already have identity, provenance state, intended role, trust state, and next action from Admission Gate.

## Visual Center

One severe central archival ledger slab labeled `SOURCE REVIEW`.

The admitted artifact appears as a small source docket attached to the left edge of the slab, already marked `ADMITTED ARTIFACT`.

The ledger slab contains exactly eight horizontal review rows:

- Source Identity
- Source Type
- Provenance Clarity
- Source Status
- Citation Relation
- Trust Limits
- Conflict / Absence Notes
- Recommended Disposition

Six disposition stamps sit along the lower edge or right margin of the ledger slab:

- Ready for Metabolization
- Needs Clarification
- Hold for Evidence
- Return to Admission
- Archive as Source
- Request Steward Review

The stamps are outcomes of review, not branches in a process flow.

Uncertainty and absence must be visible as marked review states, not hidden errors.
