## Purpose

Practice Framework gives a repeatable shape to human attention, action, reflection, and adjustment.

## Constitutional Role

It matters because The Gradient must preserve practiced judgment without converting practice into automation, certification, or doctrine.

## Boundaries

Practice Framework is distinct from Engine because it guides a human practice rather than transforming inputs mechanically. It is distinct from Registry because it is the practice object itself, not the list of practices.

## Inputs / Outputs

### Inputs

- practice intention
- human attention
- context condition
- constraint
- reflection question

### Outputs

- practice action
- reflection note
- result trace
- adjustment need
- review route

### Non-Outputs

- automation
- certification
- canon
- approval
- engine output

## Required Labels

- Practice Framework
- Practice Intention
