Object Name: Evidence.

Purpose: Evidence explains the bounded support object in The Gradient. It is material used to ground Cards, Claims, and Relationships without making them automatically true, validated, canonical, or promoted.

Constitutional Role: Evidence protects institutional reasoning from unsupported assertion. It makes support inspectable, source-linked, scoped, and reviewable. It preserves the distinction between support and proof. It allows stewards to see why a Card, Claim, or Relationship is grounded without treating that grounding as final truth.

Boundaries: Evidence follows Admission Gate, Source Review, and Metabolization Boundary. It is a derived institutional support object. It is not raw source material. It is not the same object as Source. It is not a Card. It is not a Claim. It is not a Relationship. It is not Canon. It is not Validation. It is not Promotion. It is not final proof. It does not make a Claim true by existing. It records support, scope, source linkage, trust state, and review history.

Input: Metabolized Material. The material must have passed Admission Gate, Source Review, and Metabolization Boundary.
