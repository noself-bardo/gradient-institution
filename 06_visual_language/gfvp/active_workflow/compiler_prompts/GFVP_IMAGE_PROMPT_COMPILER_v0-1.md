# GFVP Image Prompt Compiler v0.1

Status: Sandbox / Candidate Production Doctrine
Date: 2026-07-03
Package: Gradient Foundational Visual Package

## Compiler Rule

The image prompt is a compiler.

It converts a completed GFVP Object Specification into a render instruction.

It must not add new concepts, labels, diagram families, relationships, or visual metaphors that are not present in the object specification.

## Compiler Prompt Template

Render the institutional object defined by the following GFVP Object Specification.

You are depicting the object itself.

You are not depicting:

- the production process
- prompt engineering
- image generation
- QC
- IDRs
- Batch 01
- the GFVP document family
- this conversation
- software workflow
- a dashboard
- a project-management process
