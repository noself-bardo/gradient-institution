## Boundary

This roadmap is not canon by default.

It locks production scope and sequence for the sandbox visual package.

It does not approve unbuilt plates.

It does not promote any image, object, or diagram into institutional canon.

Each future plate still requires:

- object specification
- object QC
- compiled image prompt
- compiled prompt QC
- explicit image gate
- candidate generation
- output QC
- explicit approval and lock
